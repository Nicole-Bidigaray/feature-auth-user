## Rodando localmente
Clone o projeto
```bash
  git clone https://github.com/Nicole-Bidigaray/feature-auth-user.git
```

Entre no diretório do projeto

```bash
  cd my-project
```

## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
Edit .env JWT_SECRET and run this command:
# watch mode
$ npm run start:dev

Opening Database
$ npx prisma studio

## Test

```bash
